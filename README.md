
# Miinaharava

Sovellus on miinaharavapeli. Sovellus ei vaadi varsinaista rekisteröitymistä, mutta pelaajien valitsemien lempinimien avulla pidetään kirjaa heidän parhaista pisteistään Top 10:ssä.
